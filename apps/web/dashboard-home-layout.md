# Dashboard Home Layout Draft

## 목적

- 가족 홈을 이 플랫폼의 메인 허브로 정의한다.
- 공지, 일정, 투두, 진행률을 첫 진입 시 가장 빠르게 이해할 수 있게 배치한다.
- 추후 `04`, `05`, `07` 스트림이 올리는 카드가 같은 화면 구조 안에서 자연스럽게 합쳐지도록 기준을 만든다.

## 현재 기준 가정

- 첫 화면 톤은 따뜻하고 사진 친화적인 가족 홈이다.
- 첫 화면에서 가장 먼저 보여야 하는 3개 카드는 `중요 공지`, `오늘 일정`, `오늘 할 일`이다.
- 모바일 우선 설계로 시작하고, 데스크톱은 동일한 섹션을 2열 이상으로 확장한다.
- 가족 홈은 감성형이되 정보 탐색이 느려지지 않도록 "따뜻한 허브 + 빠른 확인" 균형을 유지한다.

## 섹션 구조 초안

1. `Hero`
2. `Today`
3. `Focus`
4. `Progress`
5. `Recent`
6. `Pinned`

### 1. Hero

- 가족 이름, 인사말, 오늘 날짜, 짧은 상태 메시지
- 오늘 꼭 확인해야 할 카드 1개를 강조 노출
- 가족 사진 또는 대표 배경 이미지를 사용할 수 있는 영역

### 2. Today

- 오늘 일정
- 오늘 할 일
- 긴급/중요 공지가 있으면 Today 상단으로 승격

### 3. Focus

- 오늘 안에 처리해야 하는 핵심 카드
- 읽음 확인이 필요한 공지
- 마감 임박 투두
- 오늘 시작하는 일정

### 4. Progress

- 가족 공통 목표 진행률
- 개인 루틴/습관 요약
- 연속 기록, 완료율, 주간 달성률

### 5. Recent

- 최근 글
- 최근 사진
- 최근 활동 요약

### 6. Pinned

- 관리자가 메인 고정 체크한 카드
- 기념일, D-day, 상단 고정 메시지
- 홈 성격을 유지하기 위한 장기 노출 카드

## 데스크톱 레이아웃 초안

```text
+-----------------------------------------------------------+
| Hero: 가족 이름 / 오늘 날짜 / 대표 메시지 / 최우선 카드  |
+-------------------------------+---------------------------+
| Today Schedule                | Today Todo                |
| 오늘 일정 2~3건               | 오늘 할 일 3~5건          |
+-------------------------------+---------------------------+
| Important Notice              | Focus Card                |
| 중요 공지/읽음 확인           | 오늘 처리 우선 항목       |
+-------------------------------+---------------------------+
| Progress Snapshot             | Habit / Goal Snapshot     |
| 진행률 / 퍼센트 / 단계        | 루틴 / 연속 기록          |
+-------------------------------+---------------------------+
| Recent Post                   | Recent Photo              |
| 최근 글 1건                   | 최근 사진 1건             |
+-------------------------------+---------------------------+
| Pinned / Anniversary / D-day                              |
+-----------------------------------------------------------+
```

## 모바일 레이아웃 초안

```text
[Hero]
[중요 공지 또는 최우선 카드]
[오늘 일정]
[오늘 할 일]
[진행률 또는 습관]
[최근 글]
[최근 사진]
[고정 카드 / D-day]
```

## 카드 개수 초안

### 데스크톱

- Hero: 1
- Today: 2
- Focus: 2
- Progress: 2
- Recent: 2
- Pinned: 1
- 초기 총량: 최대 10개 카드

### 모바일

- Hero: 1
- Today: 2
- Focus: 1
- Progress: 1
- Recent: 2
- Pinned: 1
- 초기 총량: 최대 8개 카드

## 카드 크기 초안

- `XL`: Hero, 긴급 공지, 핵심 가족 메시지
- `L`: 오늘 일정, 오늘 할 일, 중요 공지
- `M`: 진행률, 루틴, 최근 글
- `S`: 최근 사진, D-day, 짧은 고정 카드

## 분위기 초안

- 크림, 아이보리, 샌드, 세이지 계열을 기본 톤으로 사용
- 사진과 텍스트가 함께 살아나는 가족 앨범 같은 인상을 준다
- 과하게 장난스럽거나 유아용 같은 톤은 피하고, 가족 모두가 편하게 보는 생활 허브 느낌을 유지한다

## 사용자 확인이 들어오면 바뀔 수 있는 지점

- 첫 화면 최우선 카드 3개
- Hero 영역의 사진 비중
- 모바일에서 보여줄 카드 최대 개수
- 최근 글/사진보다 진행률을 더 위로 올릴지 여부
