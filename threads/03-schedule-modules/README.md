# 03 Schedule Modules

목표:

- 일정과 할 일 계열 모듈을 오늘 기준 홈 카드 흐름으로 정리합니다.

범위:

- `packages/modules-calendar`
- `packages/modules-todo`
- `packages/modules-school-timetable`
- `packages/modules-day-planner`

핵심 할 일:

- today 기준 카드 규칙
- 가족 공용 카드와 개인 카드 처리 기준
- 일정과 할 일의 우선순위 표현 방식

하지 말 것:

- 플랫폼 설정 화면 작업
- 인증 구조 변경

선행 조건:

- `01 Home Contract`의 계약을 따른다
