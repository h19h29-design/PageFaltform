# 00 HQ

이 스레드는 총괄 허브입니다.

## 역할

- 우선순위 결정
- 스레드 생성과 종료 판단
- 공통 계약 승인
- 충돌 가능성 점검
- 결과 취합과 통합 리뷰

## 운영 방식

- 사용자는 이 스레드에서만 지시합니다.
- 새 실행 스레드는 `threads/*/PROMPT.md` 를 첫 메시지로 시작합니다.
- 각 실행 스레드의 결과는 다시 이 스레드로 가져와 통합 판단합니다.
- 필요하면 이 스레드에서 `threads/BOARD.md` 기준으로 다음 순서를 다시 조정합니다.

## 이 스레드에서 관리할 것

- 현재 활성 스레드
- 스레드별 목표와 범위
- 공통 패키지 변경 여부
- 다음 디스패치 순서

## 공통 계약 주의 경로

- `packages/dashboard`
- `packages/modules/core`
- `packages/auth`
- `packages/database`
- `packages/platform`
- `packages/tenant`
- `packages/ui`

## 현재 상태

- 여러 개의 미니 가족 홈을 콘솔에서 직접 만들 수 있음
- 서버 파일 저장으로 내부망 공유 가능
- 내부망 테스트 주소: `http://10.137.23.25:3001`
- 다음 큰 과제는 홈 카드 계약 정리와 DB 전환 준비

## 자주 하는 지시 예시

- `01 스레드 열어`
- `01 프롬프트 줘`
- `02와 03 병렬로 시작할 패킷 줘`
- `04는 DB 전환 중심으로 좁혀`
- `05 결과 기준으로 다음 순서 정해`
